## Enable GitHub Pages (one-time, as Maliha128)

### Option A — GitHub Actions (recommended)
1. Open: https://github.com/Maliha128/Maliha-Masud-DPT/settings/pages
2. Under **Build and deployment → Source**, choose **GitHub Actions**
3. Save (workflow `.github/workflows/deploy-pages.yml` deploys the `docs/` folder)

### Option B — Deploy from branch
1. Same Pages settings page
2. Source: **Deploy from a branch**
3. Branch: `main`
4. Folder: `/docs`
5. Save
